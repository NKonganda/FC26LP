01 / Motivation

Given 18,000 players and a fixed budget,
which 11 should you pick?

Three ways to answer

  • Naïve — sort by overall rating. Ignores budget, position balance, club rules.
  • Heuristic — pick "good" players manually. Sub-optimal, slow, not reproducible.
  • LP — encode every rule as a constraint; a solver guarantees the global optimum.

Where this shows up

  • Football Manager & FC26 Ultimate Team squad building
  • Fantasy football draft tools
  • Real scouting departments with salary-cap constraints

04 / LP Formulation

Sets, parameters, variables

Sets & parameters

$I$All players ($I = 18{,}297$)
$I_{GK}, I_{DEF}, I_{MID}, I_{FWD}$Players by primary position
$r_i$Positional rating of player $i$
$p_i$Market value (€) of player $i$
$B$Budget cap

Decision variables

$$x_i \in \{0, 1\} \quad \forall\, i \in I$$

$x_i = 1$ if player $i$ is selected, otherwise $0$.

Objective

Maximise total squad rating

$$\max_{x} \;\; \sum_{i \in I} r_i \, x_i$$
05 / Constraints

Subject to

$\sum_{i \in I} p_i \, x_i \;\leq\; B$
Budget
$\sum_{i \in I_{GK}} x_i = 1$
Exactly one GK
$\sum_{i \in I_{DEF}} x_i \;\geq\; 2$
DEF floor
$\sum_{i \in I_{MID}} x_i \;\geq\; 2$
MID floor
$\sum_{i \in I_{FWD}} x_i \;\geq\; 2$
FWD floor
$\sum_{i \in I} x_i = 11$
Squad size
$x_i \in \{0, 1\} \quad \forall\, i \in I$
Integrality

Implementation: PuLP + CBC solver in Python

06 / Implementation

From math to PuLP

model += pulp.lpSum(x[i] * ratings[i] for i in range(n))           # objective
model += pulp.lpSum(x[i] * prices[i]  for i in range(n)) <= BUDGET  # budget
model += pulp.lpSum(x[i] for i in range(n) if positions[i] == 'GKP') == 1
model += pulp.lpSum(x) == 11

CBC solves the MILP; a relaxed LP run yields duals (next slide).

07 / Dual Variables

Shadow prices — what each constraint costs

Computed on the LP relaxation $(x_i \in [0,1])$. Each dual is the marginal value of relaxing its constraint by one unit.

ConstraintDualInterpretation
Budget $\leq B$$\lambda_B \geq 0$Rating gained per extra €1 of budget — the "price of cap space"
Squad size $= 11$$\mu_{squad}$Rating value of a 12th squad slot; positive → cap is binding
GK $= 1$$\mu_{GK}$Cost of holding exactly one GK vs. filling freely
DEF $\geq 2$$\mu_{DEF} \leq 0$Penalty from requiring ≥2 defenders; $0$ means non-binding
MID $\geq 2$$\mu_{MID} \leq 0$Same interpretation for midfield floor
FWD $\geq 2$$\mu_{FWD} \leq 0$Same interpretation for forward floor

$\lambda_B$ bridges the accountant's budget and the coach's ratings — it tells exactly how much squad quality each extra million buys.
